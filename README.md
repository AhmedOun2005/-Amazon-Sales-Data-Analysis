🛒 Amazon Sales Data Analysis & Dashboard
📖 Project Overview

This project focuses on analyzing Amazon sales data to extract valuable business insights and build an interactive sales dashboard for decision-making support.

The project combines:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Business Intelligence (BI)
Interactive Dashboard Development
Sales Performance Analysis

Using Python for analysis and Power BI for dashboard visualization, the project explores customer behavior, sales performance, shipping trends, product categories, and fulfillment operations.

🎯 Objectives
Analyze Amazon sales performance
Identify top-performing product categories
Understand customer purchasing trends
Evaluate shipping and courier performance
Track sales by state and fulfillment type
Build an interactive dashboard for business insights
📂 Dataset Features

The dataset contains Amazon transactional sales records including:

Feature	Description
Order ID	Unique order identifier
Date	Transaction date
Category	Product category
Size	Product size
Amount	Sales amount
Qty	Quantity sold
Ship-State	Shipping state
Fulfilment	Amazon / Merchant
Courier Status	Shipping status
Sales Channel	Amazon / Non-Amazon
B2B	Business-to-business order indicator
